# Сборка защищённой версии

## Требования

- Node.js 16+ (https://nodejs.org/)

## Установка

```bash
# Перейти в папку проекта
cd Network-map

# Установить зависимости
npm install
```

## Сборка

```bash
# Обфусцировать JavaScript файлы
npm run build
```

Результат будет в папке `dist/`:
- `dist/main.js` — обфусцированный основной код
- `dist/auth.js` — обфусцированный код авторизации

## Деплой защищённой версии

1. Создайте папку `dist/` если её нет
2. Запустите `npm run build`
3. Скопируйте в `dist/`:
   - `index.html`
   - `auth.html`
   - `styles.css`
   - `auth.css`
4. В скопированных HTML файлах пути к JS уже будут работать

## Уровни защиты

### Включено:

| Защита | Описание |
|--------|----------|
| **Обфускация кода** | JavaScript код запутан и сжат |
| **Control Flow Flattening** | Усложнение логики выполнения |
| **Dead Code Injection** | Вставка ложного кода |
| **String Array Encoding** | Кодирование строк в Base64 |
| **Запрет контекстного меню** | Правый клик отключён |
| **Блокировка DevTools** | F12, Ctrl+Shift+I заблокированы |
| **Запрет выделения** | Текст нельзя выделить |
| **Запрет перетаскивания** | Drag & Drop отключён |
| **Лицензия** | Юридическая защита авторских прав |

### Важно понимать:

Эти меры **усложняют** копирование, но не делают его невозможным. Опытный разработчик всё равно сможет получить код. Основная защита — это:

1. **Лицензия** — юридическая ответственность за нарушение
2. **Приватный репозиторий** — код не публичный
3. **Серверная логика** — критичный код на сервере (если будет бэкенд)

## Команды

| Команда | Описание |
|---------|----------|
| `npm install` | Установить зависимости |
| `npm run build` | Обфусцировать все JS файлы |
| `npm run obfuscate:main` | Обфусцировать только main.js |
| `npm run obfuscate:auth` | Обфусцировать только auth.js |
