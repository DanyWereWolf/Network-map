# Карта локальной сети

Веб-приложение для визуализации и управления сетевой инфраструктурой на базе Яндекс.Карт.

## Возможности

- **Визуализация сети** — отображение узлов, кроссов, муфт, опор и кабелей на карте
- **Трассировка жил** — отслеживание пути оптического волокна от узла до узла
- **Редактирование** — добавление, удаление и изменение объектов сети
- **Импорт/Экспорт** — сохранение и загрузка данных в JSON формате
- **Интеграция с NetBox** — импорт устройств из NetBox API
- **Авторизация** — система ролей (Администратор / Пользователь)
- **История изменений** — журнал действий пользователей

## Быстрый старт

1. Откройте `index.html` в браузере
2. Войдите с учётными данными (по умолчанию: `admin` / `admin123`)
3. Переключитесь в режим "Редактирование" для добавления объектов

## Типы объектов

| Тип | Описание | Назначение |
|-----|----------|------------|
| **Узел сети** | Конечное сетевое устройство | Сервер, коммутатор, роутер |
| **Оптический кросс** | Точка соединения волокон | Распределение/коммутация жил |
| **Кабельная муфта** | Соединительная муфта | Сращивание кабелей |
| **Опора связи** | Промежуточная точка | Прокладка воздушных линий |
| **Кабель** | Оптический/медный кабель | Соединение объектов |

## Типы кабелей

| Тип | Кол-во жил | Цвет на карте |
|-----|------------|---------------|
| Оптика 4 жилы | 4 | Красный |
| Оптика 8 жил | 8 | Оранжевый |
| Оптика 16 жил | 16 | Фиолетовый |
| Оптика 24 жилы | 24 | Бирюзовый |
| Медь (витая пара) | 4 | Серый |

## Роли пользователей

| Роль | Возможности |
|------|-------------|
| **Администратор** | Полный доступ: редактирование, управление пользователями, просмотр истории |
| **Пользователь** | Только просмотр: карта, трассировка, информация об объектах |

## Логика соединений

```
Узел сети ←→ Кросс ←→ Кабель ←→ Муфта/Кросс ←→ Кабель ←→ ...
                ↓
            Опора (транзит)
```

- **Узел** подключается только к кроссу
- **Кросс** соединяет жилы между кабелями и выводит на узлы
- **Муфта** сращивает жилы между кабелями
- **Опора** — транзитная точка, кабель проходит без изменений

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Escape` | Отмена текущей операции |
| `Delete` | Удаление выбранного объекта (в режиме редактирования) |

## Хранение данных

Данные сохраняются в `localStorage` браузера:
- `networkMapData` — объекты карты
- `networkMapUsers` — пользователи системы
- `networkMapSession` — текущая сессия
- `networkMapHistory` — история изменений

## Технологии

- **Яндекс.Карты API 2.1** — картографическая основа
- **Vanilla JavaScript** — без фреймворков
- **CSS3** — адаптивный дизайн
- **LocalStorage** — локальное хранение данных
