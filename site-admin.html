Организации

  • Полный список организаций из хранилища сервера: тариф (planId), лимит сессий, дата подписки, контактная почта.
  • «Настроить» переключает на вкладку «Тарифы и лимиты» с уже выбранной организацией.
  • «Удалить» безвозвратно удаляет организацию, карту, историю и пользователей этой организации.
  • Роли и список пользователей по-прежнему ведутся в приложении карты (модальное окно «Пользователи»).

Тарифы и лимиты

  • Тариф организации (planId) — встроенные basic / pro / enterprise и id карточек с вкладки «Витрина тарифов»; лимит сессий и дату подписки можно переопределить отдельно.
  • Приостановка (приостановлена) сбрасывает сессии и блокирует вход для всех пользователей организации.
  • Ниже — сброс пароля пользователя организации (завершает его активные сессии).
Витрина: вкладка «Витрина тарифов». Если у организации нестандартный id, он появится в списке как «Текущий».
0 или пусто — использовать лимит из базового плана.
Пусто — без ограничения по дате (под вашим контролем).
Приостановленная организация не сможет войти, даже при оплате.
Указывается при регистрации; можно изменить здесь.

Сброс пароля пользователя

  • Выберите организацию и учётную запись; новый пароль действует сразу, старые сессии этого пользователя завершаются.

Витрина тарифов (pricing.html)

  • Карточки на публичной странице pricing.html и на корне сайта (если открыт лендинг) — только отображение и кнопки.
  • Числовой лимит в карточке подхватывается для организаций без ручного лимита, если ID тарифа совпадает с planId организации.
  • Акции и скидки: процент и даты; опционально «цена до скидки» для зачёркивания.
Название Цена Лимит Скидка Порядок Тип Действия
Целое ≥ 1 или −1 (без ограничения). Пусто — брать лимит из встроенных basic/pro/enterprise по ID.
Пока даты действуют и указан процент скидки, на странице тарифов показывается бейдж и при необходимости зачёркнутая «старая» цена.

Подписки

  • Показаны только организации с датой окончания подписки в пределах выбранного горизонта (дней) или уже с истёкшей датой.
  • Кнопки +30 дней и +1 год сдвигают дату окончания от текущей даты окончания (или от сегодня, если истекла).
  • После продления пользователи снова смогут входить, если организация не приостановлена.
