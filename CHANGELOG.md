# История изменений

- Визуализация сетевой инфраструктуры на Яндекс.Картах
- Объекты: узлы сети, оптические кроссы, кабельные муфты, опоры связи
- Прокладка кабелей (оптика 4/8/16/24 жилы, медь)
- Соединение жил в кроссах и муфтах, подключение жил к узлам
- Группы кроссов и узлов, трассировка жил
- Поиск по карте, экспорт/импорт JSON, импорт из NetBox
- Авторизация и роли (администратор / пользователь)
- История изменений, тёмная и светлая тема
- Справка по программе, проверка обновлений через GitHub
