# Идеи по доработке

- **Разбиение main.js** — вынести логику по модулям (карта, объекты, кабели, GPON, справочник устройств, undo/redo и т.д.) с подключением через отдельные скрипты или сборку.
- **Большая карта** — при большом числе объектов: отрисовка только видимой области и/или упрощённые маркеры при маленьком зуме (LOD) для снижения подвисаний.
- **Экспорт в изображение** — сохранение текущего вида карты в PNG/SVG.
- **Печать** — печать выбранной области или всей карты с легендой.
- **Поддержка нескольких карт** — переключение между несколькими проектами карт на сервере.
- **Роли** — расширение ролей (например, редактор только своих объектов, просмотр по зонам).
